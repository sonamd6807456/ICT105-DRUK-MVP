# Reflection

### Which feature creates the strongest user value?

The **Event Listing, Search & Filter** feature creates the strongest user value because it allows students to quickly discover relevant campus events without searching across multiple communication channels. This directly addresses the main problem identified during customer discovery.

### Which feature creates the strongest business or operational value?

The **Admin Event Management** feature creates the strongest operational value because it enables organizers and administrators to create, update, approve, and manage events efficiently. Keeping event information accurate and up to date improves communication and increases student participation.

### Which required feature is still weak or unclear?

The **Dashboard & Analytics** feature is currently less developed because the prototype uses sample data rather than real-time event statistics. In future versions, it could be enhanced by integrating a database to provide live analytics and participation reports.

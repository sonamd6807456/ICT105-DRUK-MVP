# Screen Inventory Checklist

| Screen ID | Screen Name | Required? | Requirement IDs | Has Wireframe? | Has Screenshot? | Notes |
|---|---|---|---|---|---|---|
| S-01 | Homepage / Landing Page | Yes | FR-01, FR-02 | No | No | Entry point for students and organizers. Shows Smart Campus Event Hub introduction, upcoming events, navigation menu, Browse Events, Register, and Login actions. |
| S-02 | Event Registration Form | Yes | FR-05, FR-06 | No | No | Allows students to register for campus events by providing user information and receiving registration confirmation and reminders. |
| S-03 | Event Information List | Yes | FR-01, FR-03, FR-10 | No | No | Displays available campus events with search, filters, categories, dates, organizers, and event status information. |
| S-04 | Event Detail View | Yes | FR-04, FR-05, FR-08 | No | No | Shows complete event information including description, venue, schedule, organizer, registration status, and calendar option. |
| S-05 | Student Dashboard / Event Summary | Yes | FR-06, FR-07 | No | No | Provides overview of registered events, upcoming activities, notifications, reminders, and participation history. |
| S-06 | Organizer / Admin Management View | Yes | FR-02, FR-09, FR-10 | No | No | Allows organizers to create, edit, delete, publish events, manage registrations, and update event status. |

## Rule

A screen that is required in Lab 05 must be carried forward into the final prototype unless the team documents an approved change in GitHub issues and weekly logbook.

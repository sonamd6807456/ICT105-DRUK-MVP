# User Consent Statement

This prototype has been developed for the **ICT105 Fundamental Technology Entrepreneurship** course as part of the **Smart Campus Event Hub** project.

Users are invited to test the prototype by completing common tasks such as creating an account, browsing campus events, viewing event details, registering for events, saving favorite events, and providing feedback.

Participation in this usability test is **completely voluntary**. The information collected will be used **only for educational purposes**, including prototype evaluation, usability analysis, project improvement, and final course reporting.

Users should **not provide sensitive personal information** beyond the information required to test the prototype. During testing, sample or fictional data may be used to protect user privacy.

---

## Data Collected

The prototype may collect the following information during usability testing:

- Full Name or Nickname (optional)
- Student ID (sample or test data)
- Faculty
- Email Address (sample or test data)
- User Role (Student, Organizer, Administrator)
- Task Completion Result
- Ease of Use Rating (1–5)
- Usefulness Rating (1–5)
- Interest Level (1–5)
- Feedback Comments
- Suggestions for Improvement

---

## Data Use

The collected information will be used only for:

- Evaluating the usability of the Smart Campus Event Hub prototype.
- Measuring task completion and user satisfaction.
- Identifying usability issues and improvement opportunities.
- Supporting Lab 09 documentation and analysis.
- Improving future versions of the prototype.
- Preparing the final ICT105 project report.

The collected information will **not** be shared with third parties, sold, or used for commercial purposes. Personal information will remain confidential and will only be accessible to the project team and course instructor for academic evaluation.

Participants may stop testing at any time without any penalty or negative consequences.

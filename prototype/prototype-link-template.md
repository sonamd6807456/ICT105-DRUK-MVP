# Prototype Link

## Prototype Platform

- Platform: HTML, CSS, and JavaScript frontend web prototype hosted on GitHub Pages
- Prototype link: https://github.com/sonamd6807456/ICT105-DRUK-MVP
- Access note: The prototype is publicly accessible through the GitHub Pages link. Users can browse events, search and filter event listings, view event details, register for events, and access student and organizer/admin functions.
- Last updated: 2026-07-21

## Requirement Coverage

| Requirement ID | Where It Is Shown in the Prototype |
|---|---|
| FR-01 | Smart Campus Event Hub homepage showing the system name, purpose, target users, and main navigation such as Events, Register, Dashboard, and Profile. |
| FR-03 | Admin Dashboard and Manage Events page where organizers or administrators can create and submit event information including title, description, date, time, location, category, and capacity. |
| FR-05 | Events page showing available campus events in a structured card/list layout. |
| FR-06 | Events page with search and category filtering functionality for finding relevant campus events. |
| FR-07 | Event Details page showing full event information including description, date, time, location, organizer information, and available seats. |
| FR-08 | Event Details page, registration interface, and Student Dashboard showing event registration and participation status. |
| FR-09 | Admin Dashboard and Manage Events page for managing, editing, approving, updating, or removing event information. |
| FR-12 | Student Dashboard and Admin Dashboard showing event and participation summaries such as upcoming events, registered events, event statistics, and management information. |
